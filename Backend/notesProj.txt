In .env File use this variables: 
    DB_HOST = ''
    DB_USER = ''
    DB_PASSWORD = ''
    DB_DATABASE = ''
    SECRET = ''
    NAME = 'UserNamePC'

In DB use this commands: 

CREATE DATABASE cashSystem;
USE cashSystem;
